# AxeView

자바스크립트에서 뷰모델(View-Model) 형태로 HTML을 다룰 수 있게 해주는 라이브러리입니다.

HTML에서 지정한 돔(dom)을 지정하고 연결될 데이터를 표시 뷰(View)전용 라이브러리입니다.

돔과 뷰모델(View-Model)을 연결하여 관리해줍니다.  
뷰모델에 접근하여 데이터를 쉽고 빠르게 수정하고 View에 반영할 수 있습니다.
