출력 타입 변경

이미 생성된 감시자의 출력 방식(OverwatchingOutputType)을 변경할 수 있습니다.
Overwatch.OutputTypeChange_All(OverwatchingOutputType)를 통해 동적으로 타입을 바꿀 수 있습니다.

단, 출력 타입을 변경하면 연결 되있던 돔(DOM)이 다시 생성되기 때문에 기존에 연결되어 있던 주소(변수 등등)가 끊기게 됩니다.
감시자는 자동으로 변경된 돔을 감지하므로 사용에 지장이 없습니다.

* 감시방식 변경은 기능을 넣을지 말지 확정되지 않았습니다.
변환 가능 관계
String <-> Html <-> Dom


1) 노드 변경
String <-> Html <-> Dom 간의 변경이 가능합니다.

*Html, Dom입력은 자동으로 h3으로 생성됨
