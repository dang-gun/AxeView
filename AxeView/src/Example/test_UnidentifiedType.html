불확실 타입

감시자를 생성할때 자동으로 감시방식이 지정되도록 OverwatchingType.Unidentified을 사용할 수 있습니다.
이 타입을 쓰면 AxeView가 자동으로 판단하여 감시방식을 변경합니다.

이 타입은 'AxeViewSupport' 라이브러리를 위해 만들어졌습.
직접 사용하기보다는 뷰를 기준으로 감시자가 없는 대상을 생성할 때 사용하는 것이 좋습니다.



1) 텍스트 노드(Text Node)

테스트 :
front ▶▶▶ {{TextNode_UnType}} ◀◀◀ back





2) 속성(attribute) - 값이 있는 속성
값이 있는 속성은 무조건 OverwatchingType.Monitoring으로 동작합니다.

테스트 :
속성 테스트용 돔입니다.
개발자 툴을 열어 변경된 속성을 확인하세요~




3) 속성(attribute) - 이름만 있는 속성
이름만 있는 테스트 돔1입니다.



4) 속성(attribute) - Value 감시

입력값 출력 :



5) 속성(attribute) - Event 감시
속성 이름이 'on'으로 시작되면 무조건 이벤트라고 판단합니다.
'OverwatchingType'은 'Monitoring'으로 교체됩니다.
'OverwatchingOutputType'은 'Function_NameRemoveOn'으로 교체됩니다.
